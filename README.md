# gpt-short-maker

# GPT 를 통해서 쇼츠를 자동 생성하는 파이썬 코드입니다
1. chat gpt 를 통해 대본을 생성합니다
2. 이에 대한 음성 TTS 를 생성합니다
3. 음성 TTS 에 맞는 이미지를 chat gpt 를 이용해 생성합니다
4. 이걸 python 의 moviepy 라이브러리를 이용해 다 합쳐서 짬뽕 시킵니다
